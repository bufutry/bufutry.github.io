7.22-7.26复盘

发表于 2019-07-28 | 分类于 投资理财

7月22日

    基本情况：
两连阳：144 上涨：351 下跌：3261 热点：醋酸、新能源汽车、军工、智能

7月23日

    基本情况：
两连阳：176 上涨：2950 下跌：523 热点：军工、新能源汽车、互联网。

7月24日

    基本情况：
两连阳：2226 上涨：3050 下跌：459

7月25日

    基本情况：
上涨：1819 下跌：1558

7月26日

    基本情况：
两连阳：711 上涨：1624 下跌：1788 热点：自贸区、苹果概念、新能源汽车。

小结：周一科创板上市交易，盘中走出牛股安集科技翻了5倍，科创板首日25只个股全部翻倍，前5日的不设置涨跌幅，T+1模式交易，之后单日涨跌幅设置为20%。周一科创板交易成交高达450亿，主板指数大跌又有所谓的老师出来说因为科创板抽血主板就跌了，可是在上周5主板涨的时候他们却说科创板要上了面子工程国家队会护盘的，周一肯定会涨。笑死老夫了。所谓的逻辑都是事后找到一个让自己不承认自己错了的理由。周一再说抽血大跌的结果主板周2就开始反弹了，本周一波4连阳。但是也不是太好赚钱情绪分歧比较大，很多资金都是在观望，但是在这情绪中走出了4连阳，虽然4连阳但是外资和大盘的资金都表现出流出。下周开始要一波补跌了，科创板也结束了不设涨跌幅的限制了，算是所谓政治任务面子工程完成了吧。不得不佩服制定科创板涨跌幅的神人，科创板首日后，平均涨跌幅确实在20%附近波动。这段时间有点短线情绪和大盘情绪共振的感觉，短线情绪不独立于大盘情绪了，顺势而为才行短线投机时大盘情绪也要权重放大了。下周就到了八一建军节，这段时间其实军工一直在异动像一条暗线。这个周末中美将在上海就贸易战重新谈判，周五上海自贸区有所异动了，看看周末消息落地周一发现情况吧。对于市场的因为T+1的规则制约，单纯的跟随在行情不好时是不好做的。应该是对市场进行预判，然后做出相应的应对计划，应对比预测更为重要，至于结果那就是交给市场了。
